メニュー作成

現在のウィンドウにリンクするには、location.href = "URL" を使う。
新しくウィンドウを開いてリンクするには、window.open( "URL" ) を使います。
別ウィンドウでの表示 　window.open('URL','ウィンドウ名','属性');　属性とはウィンドウの横縦幅やスクロールバーのありなしのこと
window.open( )で小窓という意味　htmlでは、イベント属性のonclick=""を記載。　「javascript:void(0)」（何もさせない）の意味 外部リンク参考サイトさん 別ウィンドウでの表示

PW設定

  • PW設定基礎　 外部リンク参考サイト PW設定
    • prompt　prompt(　) は、入力ウィンドウを表示します。大文字/小文字は区別されます！

      • 　prompt(　)でインプットボックスを出します。 括弧内は２つの区分があります。
      最初の部分は質問文を記入し、後ろの部分は入力枠に表示する文字を指定      します。
      今回は入力枠に何も表示する必要が無いので、空にしています。
      文字列なのでクォーテーションで囲み、２つの区分の間は半角コンマで区切ります。
      変数 = prompt("質問文","最初に表示する文字");
      カッコ内の最初の項目は質問文、2番目の項目は初期表示文字を指定します。
      if～else文　条件に合っているかどうかによって、別々の処理を行う方法

      if ( 条件式 ){ 処理1 } else { 処理2 }

      (条件式)に合致している場合、{処理1}を実行し、 条件式に合致しない場合、{処理2}を実行します
    • PW設定ややセキュリティアップ　String.fromCharCode(　) は、文字コードを文字に変換。
      ひとつひとつの文字には文字コードが割り付けられています。コンピュータは文字ではなく 文字コードで処理しています。
      String関数を使い、 String.fromCharCode(　) にして、文字コードを私たちが分かる文字へ変換します。

    切り替えなど

    マウスでの画像切り替え

    「何もしていない最初の画像」、「マウスを乗せた時」、「マウスを離した時」
    の３つの画像ファイル名を３つとも ＜IMG src=＞の中に記載。

    それぞれ=から先を”（ダブルクォーテーション）で囲む。
    　onMouseOver="this.src='画像ファイル名'"　onMouseOut="this.src='画像ファイル名'"　
    　マウスオーバー、アウトの場合のみ="this.src='画像ファイル名'"    にする
    「 scrollTo( x, y ) 」 　 body 要素を座標 ( x軸, y軸 ) にスクロールさせる。なのでscrollTopだとトップへ戻る
