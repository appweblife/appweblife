windowオブジェクト

openメソッド 新しいウィンドウを開きます。

window.open( "URL ", ウィンドウ名" , "オプション")

カッコ内に、"表示させるページのURL"と次に"ウィンドウ名"を指定し、オプションで、"サイズや位置"などを指定します。
それぞれクォーテーションで囲います。

width=数値　ウィンドウの幅を指定。
height=数値　ウィンドウの高さを指定。
top=数値　PC画面上端からの距離。
left=数値　PC画面左端からの距離。

documentオブジェクト

Windowオブジェクトの一部。
「document.プロパティ」や「document.メソッド」のように「.（ドット）」と共に使うことが多い。
document.write(文字);

文字をページに表示する

document.cookie

クッキーを習得する。ログイン情報など。

document.getElementById(　)

HTMLのid要素のテキストを取得するメッソド。
document.getElementById("id要素名");

innerHTMLで要素を変える。

XMLHttpRequestオブジェクト

略して[XHR]

HTTPを利用してWeb上のファイルを取得することができる。ページ遷移なしに更新、読み込むことができる。

[new演算子]を使いインスタンスを作成する。

var変数 = new XMLHttpRequest();
