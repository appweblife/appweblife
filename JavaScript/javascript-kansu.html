引数

ユーザー定義関数

宣言した関数は、関数名()と記述することで再度使用できます。

function 関数名(引数){ 命令文}

functionを記述します。functionの後に半角スペースを入れて関数名を記述。
関数名の後に括弧(　)を付けます。この中には「引数」というものがはいるが、引数を使わない場合は空にする。
引数が2つ以上必要な場合もあり、その場合、「,」で区切る

{　}の中に命令文を記述。　 。

コールバック関数

引数が関数になる。

無名関数(関数リテラル)

関数名を記述しない。関数名がなくてよい。
コードが見やすくなるメリットがある。

組み込み関数（ビルトイン関数）

new演算子を使ってインスタンスを作成し、使います。

Array （配列を扱う）Array(　)の引数に配列の要素を記述。　
Date （日付を扱う） new演算子を使ってインスタンスを作成する

var 変数名 = new Date(　);

パラメータ（引数）なし、 現在の日付、時刻。
