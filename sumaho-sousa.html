音楽

PCからandroidへ

幾つか方法があります。CDなどの音楽データをmp3形式等にする必要があります。
直接フォルダ ＊機種によって表示が異なります。
  1. USBケーブルでWindowsとandroidを接続。
  2. Android端末に表示される 「ファイル転送」にチェック
  3. PCの「MUSIC」フォルダからスマートフォンの「内蔵ストレージ」→「MUSIC」フォルダに直接ドラッグ＆ドロップ。
WindowsMediaPlayer ＊機種によって表示が異なります。
  1. USBケーブルでパソコンとandroidを接続。
  2. Android端末で表示される 「ファイルを転送」にチェック
  3. WindowsMediaPlayerの左側に表示される転送するandroid端末を選択
  4. 右側「同期」タブをクリック
  5. 左側「音楽」または「プレイリスト」をクリックし楽曲を右下「同期リスト」へドラッグ＆ドロップ
  6. 右側「同期の開始」をクリック。

ファイルマネージャー

Bluetooth転送 ”androidのみ”。iPhoneでは出来ません。転送速度遅め。
あらかじめandroidとWindowsでペアリングしておき、Android、Windows両方、BluetoothをONにします。
Windows設定
  1. パソコン画面右下”Bluetoothのアイコン”をクリック（表示されてない場合は△マークをクリック）
  2. [ファイルの受信]をクリック
  3. 「接続の待機中」の表示が現れます
  4. androidからファイルが送信され受信すると「受信したファイルの保存」の表示が現れます
  5. 「場所」から「参照」をクリックし保存先を選択します（デフォルトでよければそのままでOKです）
  6. 完了をクリックします
android設定 ※機種により異なります
  1. 送信したいファイルを選択
  2. 「共有」をクリック
  3. ［Bluetooth］をクリック
  4. 「デバイスを選択」の表示になり送信するWindowsを選択します

iPhoneからWindows10へ

幾つか方法があります。
  1. Windows10とアイフォンをLightningケーブルで接続します。
  2. iPhone上に表示される「このデバイスに写真やビデオへのアクセスを許可しますか？」 の表示の許可にタップ
  3. PC上に表示されるiPhoneをクリック
  4. 「Internal Storage」
  5. 「DCIM」
  6. 「100APPLE」などのフォルダ内に画像があります
フォトアプリ
  1. Windows10とアイフォンをLightningケーブルで接続します。
  2. Windows10でフォトアプリを起動します。
  3. アイフォン
  4. フォトアプリ右上「インポート」
    5. 枚数が多いと時間がかかります。 一括選択、または月ごと選択ができます。
