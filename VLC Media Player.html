概要

Windows、Mac、Linuxでも使用可能な”動画再生ソフト”。広告も無く無料で使用できます。
動画の早送りやスロー再生、色調調整、MP４など動画の変換、修復も出来ます。

動画再生ソフトのため動画保存には向いていません

動画の再生

再生の基本

再生
  1. 開きたい動画ファイルを"右クリック"
  2. 「プログラムから開く」
  3. 「VLC media player」をクリックします。
シークバー

動画、音声データの再生位置を表示する機能のこと

スライダーがバーの左から右へ移動してゆく
動画をダブルクリックで再生

毎回「プログラムから開く」からではなく、ファイルをダブルクリックすればすぐに再生できるように設定します

いろいろな再生

リピート再生 ズーム再生

※スマホで撮影した動画の場合、うまくズーム出来ないことがあります
  1. 左下「詳細設定」のアイコンをクリック
  2. 「ビデオエフェクト」
  3. 「ジオメトリ」を選択
  4. 「インタラクティブズーム」にチェックをいれます。
  5. 左上に小さくウィンドウができ、ウィンドウの中の白枠がズーム箇所になります。
    • "白枠"をドラッグすることによりズームする位置を調整できます。
    • ウィンドウの下にある逆三角形をドラッグしてズーム箇所を拡大縮小できます
再生速度調整
右下の”速度表示”をクリックし、目盛りを左右にドラッグすると速度調整できます
再生速度ショートカット
  • 「再生速度アップ」　+
  • 「再生速度ダウン」　-
  • 「再生速度0.1アップ」　]
  • 「再生速度0.1ダウン」　[
  • 「コマ送り」　e
速度セレクター メニューバーの「ツール」－「インターフェースのカスタマイズ」
コマ送り

ショートカット「e」。または左下”縦線とやじるしの形のコマ送りのアイコン”をクリック

回転

90度回転、180度回転や細かな角度での回転などいろいろ調整できます

  1. 右クリック「ツール」
  2. 「エフェクトとフィルター」
  3. 「ビデオエフェクト」

  4. 「ジオメトリ」
    • 「変換」にチェックをいれ、回転、反転など任意のものを選択
    • 細かく設定する場合は「回転」にチェックを入れアンクルを動かし調整します
アスペクト比

画面の縦横比の設定

ショートカットキー「A」
  1. 右クリック「ツール」
  2. 「ビデオ」
  3. 「アスペクト比」
  4. 任意の数値を選択
●アスペクト比4：3

●アスペクト比16：9

クロッピング
  1. 右クリック「ツール」
  2. 「エフェクトとフィルター」
  3. 「ビデオエフェクト」
  4. 「クロッピング」
  5. 上下左右任意の数値を入力。必要に応じて上下左右連動にチェック
クロッピングした動画を保存

色調調整

明るさ、コントラスト、彩度などの調整

  1. 右クリック
  2. ツール
  3. 「エフェクトとフィルター」
  4. ビデオエフェクト
  5. 基本
  6. 「明るさ」「コントラスト」など各項目の調整

複数起動

動画を2画面、３画面など複数再生起動できます
  1. 右クリック
  2. ツール
  3. 設定
  4. 「インターフェース設定」タブの「プレイリストとインスタンス」項目にある下記のチェックを外す
    • 「ひとつだけ実行を許可」
    • 「ファイルマネージャーから起動された場合、単一インスタンスで実行」
  5. 「保存」をクリック

録画

設定 右クリック

録画方法

動画を録画

動画を再生中に左下”赤いボタン”をクリック

画面を録画
  1. 「VLC」を起動させます
  2. 右クリックし「メディアを開く」（またはショートカットキーCtrl+C）
  3. 「キャプチャーデバイスを開く」を選択
  4. 「キャプチャーデバイス」タブの「キャプチャーモード」で「デスクトップ」を選択
  5. 「フレームレート」を設定
    ※上げると滑らかになりますが容量が増えます、下げると画質が落ちますが容量は減ります 。30.00フレーム程度に設定。滑らかに録画したいのであれば60.00フレーム。
  6. 「変換」を選択しクリック
  7. 「設定」「プロファイル」横のアイコンをクリック
  8. 「プロファイルのエディション」が開きます
  9. 「ビデオコーデック」タブ
  10. 「エンコーディングパラメーター」の「ビットレート」を「6000kb」ぐらいが適正
  11. 「保存」をクリック。
  12. 「参照」をクリックし録画ファイルの保存先とファイル名を決定します
  13. 「開始」をクリックし録画スタート
  14. 「停止」をクリックすると録画終了

動画の変換

MP４などに変換できます

  1. 「メディア」
  2. 「変換/保存」
  3. 「ファイル」
  4. 「追加」
  5. 修復したいファイルを選択
  6. 変換を選択しクリック
  7. 参照をクリックし保存先を決定します

動画の修復

VLCで”再生出来ない場合”は修復は難しい

  1. 「メディア」
  2. 「変換/保存」
  3. 「ファイル」
  4. 追加
  5. 修復したいファイルを選択
  6. 「オリジナルのビデオトラックを保持」にチェックを入れる
